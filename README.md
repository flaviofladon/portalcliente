# Portal do Cliente - WordPress Theme

Tema WordPress custom desenvolvido para o **Portal do Cliente**.

## Tecnologias

- WordPress
- PHP
- CSS (custom)
- JavaScript
- Roboto Font

## Estrutura

assets/
css/ → estilos do tema  
js/ → scripts do tema  

## Desenvolvimento

Modo desenvolvimento:

WP_DEBUG = true

Usa:

main.css  
main.js

Produção:

WP_DEBUG = false

Usa:

main.min.css  
main.min.js

## Instalação

1. Copiar o tema para:

wp-content/themes/

2. Ativar no WordPress


## Plugins necessários:

- Advanced Custom Fields (acf obrigatório)
- Yoast
